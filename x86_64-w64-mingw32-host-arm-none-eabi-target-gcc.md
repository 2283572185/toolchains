# GCC14工具链

## 平台

| build            | host               | target        |
| :--------------- | :----------------- | :------------ |
| x86_64-linux-gnu | x86_64-w64-mingw32 | arm-none-eabi |

## 版本

- GCC：14.0.1
- GDB：15.0.50
- Binutils：2.42.50

## 组件

- gcc
- g++
- binutils
- gold
- gdb (需要python3.11)
- Python3.11.6 embed package
- .gdbinit (位于share下)
- 调试符号：libgcc libstdc++
