# GCC15工具链

## 平台

| build            | host             | target     |
| :--------------- | :--------------- | :--------- |
| x86_64-linux-gnu | x86_64-linux-gnu | x86_64-elf |

## 版本

- GCC：15.0.0
- GDB：16.0.50
- Binutils：2.42.50

## 组件

- gcc
- g++
- binutils
- gold
- gdb (需要python3.12.3)
- Python3.12.3 embed package
- .gdbinit (位于share下)
- 调试符号：libgcc libstdc++
