# GCC15工具链

## 平台

| build            | host             | target        |
| :--------------- | :--------------- | :------------ |
| x86_64-linux-gnu | x86_64-linux-gnu | arm-none-eabi |

## 版本

- GCC：15.0.0
- GDB：16.0.50
- Binutils：2.42.50

## 组件

- gcc
- g++
- binutils
- gdb (需要python3.13.2)
- .gdbinit (位于share下)
- 调试符号：libgcc libstdc++
